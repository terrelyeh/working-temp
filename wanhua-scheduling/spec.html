1. 硬性限制 HARD

排班系統絕對不能違反；違反即視為非法排班。

H1. 工時上限

  • H1.1 固定 PT 月工時 ≤ 90h
  • H1.2 不固定 PT 月工時 ≤ 36h
  • H1.3 主管月支援工時 ≤ 40h 🔴 由「建議」改硬性，可由設定切換
  • H1.4 正職月工時 ≤ 月上限（依當月需求表）

H2. 班次間休息 🔴 新增（依勞基法）

  • H2.1 同一員工兩班次之間休息 ≥ 11 小時
    • 推論：長班日晚班 00:30 下班 → 下次最早 11:30 上班，因此不能排隔日早班（09:00）也不能排午班（11:00）。
    • 修正 v1「前一天晚班員工不可排隔日早班」→ 連午班也不行，且全週適用。

H3. 連續工作天數 🔴 補齊混排規則

  • H3.1 連續工作天數 ≤ 6 天（任何日數，週一公休自然限制）
  • H3.2 純 8h 班次連續 ≤ 5 天
  • H3.3 含 10h 班次的連續工作 ≤ 2 天（任一連 2 日內若兩天都有 10h 班，視為觸頂）
  • H3.4 跨月計算：12/30、12/31、1/1、1/2 視為連續

H4. 關鍵時段最低人力

  • H4.1 平日：每個班至少 1 位正職（午班與支援班可共用同一位）
  • H4.2 長班日：早班 ≥ 2 正職、晚班 ≥ 1 正職
  • H4.3 週日：早班 ≥ 1 正職、午班 ≥ 1 正職
  • H4.4 國定假日（非週一）：依該日對應的日別規則 + 至少 1 位正職

H5. 同日多班 🔴 新增

  • H5.1 同一員工同一日只能排一個班次（禁止 split shift）
    • 🟡 待決：是否開放例外？

H6. 主管使用限制

  • H6.1 主管不可佔據「該日仍有正職可排」的班次
  • H6.2 「所有正職都無法排」= 當天可排班正職皆已排滿、休假、或請假 🔴 釐清原文歧義

2. 軟性限制 SOFT

應盡量達成；無法達成需記錄理由並由主管核准。

S1. 工時目標

  • S1.1 正職月工時達到目標（依需求表）
  • S1.2 固定 PT 月工時 ≥ 56h
  • S1.3 固定 PT 月工時 ≤ 70h（預警線；超過需理由）
  • S1.4 固定 PT 月工時 ≤ 80h（警戒線；超過需主管核准）

S2. 公平性指標 🔴 新增（原章節有名無實）

  • S2.1 員工間「晚班次數」標準差 ≤ 🟡（建議 1.5）
  • S2.2 員工間「長班日上班次數」標準差 ≤ 🟡
  • S2.3 同一員工不應連續多週都被排到不喜歡的班（需記錄員工偏好）

S3. 班次配置偏好

  • S3.1 有 10h 能力的正職優先排 10h 班（達標效率高）
  • S3.2 國定假日優先安排正職（基於正職享國假加倍薪資的權利）🔴 重寫原文邏輯

3. 偏好 PREFERENCE

員工個人設定，影響排班但可被覆蓋。

  • P1. 不可上某類班次（如：不能上晚班、不能上週末）
  • P2. 偏好班次（早班 / 午班 / 晚班）
  • P3. 希望休假日（每月最多 N 天可指定）

🟡 待決 偏好違反的優先級。建議：硬偏好（P1）視為硬性限制，軟偏好（P2/P3）視為軟性。

5. 排班流程

5.1 月初準備

  1. 接收當月排班需求摘要表
  2. 計算可行性
    • 每位正職最大可能工時 = (該月營業日 - 該員工不可排日數) × 該員工最大日工時
    • 緩衝率 = 目標工時 ÷ 最大可能工時
    • 緩衝率 ≤ 0.95：可達成
    • 0.95 < 緩衝率 ≤ 1.00：高風險（需協調）
    • 緩衝率 > 1.00：不可達成（必須調整目標或休假）
  3. 蒐集員工偏好與請假需求

5.2 排班優先序

  1. 正職達標（含 10h 班、關鍵時段）
  2. 關鍵時段補位（H4 各條）
  3. 固定 PT 達 56h
  4. 不固定 PT 填缺（不超過 36h）
  5. 主管支援（不超過 40h）

5.3 排班結果驗證（最終檢查）

所有 H1–H6 硬性限制必須通過；軟性限制違反需列出理由清單。

6. 異常處理流程 🔴 新增

6.1 臨時請假

  1. 系統標記該員工該日為「不可排」
  2. 重新驗證受影響日的關鍵時段人力是否仍達標
  3. 不達標 → 依以下順序找代班：
    1. 同類別、當日休假且未排滿月工時的員工
    2. 不固定 PT（若未達 36h 上限）
    3. 主管支援
  4. 通知相關員工，記錄代班補償

6.2 月中入職 / 離職

  • 工時目標按比例調整：調整後目標 = 原目標 × (在職天數 ÷ 當月總天數)
  • 入職首週不安排晚班（🟡 待決）

6.3 颱風 / 天災停業

  • 該日不計入連續工作天數
  • 工時目標按比例下修
  • 員工出勤費用依公司政策（🟡 待決）

6.4 國定假日特殊規則

  • 非週一國假：照常營業，正職享加倍薪資
  • 週一遇國假：仍公休，不補假（🟡 確認）
  • 國假計入正職休假天數配額
